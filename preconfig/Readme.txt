Подготовка к выполнению run_playbook.sh:
- Установить на целевую машину open-ssh 'sudo apt-get install ssh'
- Применить к целевому хосту '../user_ubuntu', предварительно указав адрес удаленного хоста для настройки
- Отредактировать inventory
- При необходимости, закоментировать ненужные роли

P.S. Приватный ключ пользователя 'Ubuntu' должен располагаться по пути, указанном в файле inventory.
